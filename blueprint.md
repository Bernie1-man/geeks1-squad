# **App Name**: GeekForce Central

## Core Features:

- User Authentication: Secure login/signup using email/password, allowing all 9 members to register and log in.
- Dashboard: Personalized dashboard with member profiles, status, and dark mode toggle.
- Task Management: Assign and track tasks for each member, with notifications for updates.
- Group Chat: Real-time chat for team collaboration and communication.
- Event Calendar: Plan Geek Squad activities or missions with an integrated event calendar.
- Profile Customization: Allow users to update their profile picture and description. Leverage AI to suggest bios and profile pics.
- Activity Insights: AI tool that uses task assignments and calendar events to create brief summaries of recent or expected activity. Summaries include suggested next steps or potential conflicts.

## Style Guidelines:

- Primary color: Deep midnight blue (#2C3E50) to convey a professional and reliable image.
- Background color: Dark slate gray (#34495E) for a sleek dark mode interface.
- Accent color: Electric blue (#3498DB) for interactive elements and highlights.
- Body and headline font: 'Space Grotesk' sans-serif for a modern and techy feel. Note: currently only Google Fonts are supported.
- Use sleek, modern icons representing different tasks and activities.
- Fully responsive layout for seamless experience on both desktop and mobile devices.
- Subtle animations for transitions and interactions to enhance user experience.
